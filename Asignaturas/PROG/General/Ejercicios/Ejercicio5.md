# 🍎 Ejercicio 5: Supermercado Interactivo
**Objetivo:** Crear un programa que permita gestionar un inventario de productos de forma interactiva.

## Instrucciones:
1. Crea una clase `Producto` con los atributos `nombre` y `precio`.
2. Crea una clase `Supermercado` que tenga:
    * Una lista vacía llamada `almacen`.
    * Un método `añadir_producto` que reciba un objeto y lo guarde en la lista.
    * Un método `mostrar_inventario` que recorra la lista y muestre los datos.
3. Implementa un programa interactivo que:
    * Permita añadir productos solicitando su nombre y precio.
    * Muestre el inventario después de cada acción.
    * Finalice cuando el usuario lo indique.
