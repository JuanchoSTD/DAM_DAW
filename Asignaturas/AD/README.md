# ASIGNATURA DE ACCESO A DATOS.

Este módulo profesional contiene parte de la formación necesaria para desempeñar la función de desarrollador de aplicaciones multiplataforma.

La función de desarrollador de aplicaciones multiplataforma incluye aspectos como: el desarrollo de aplicaciones que gestionan ficheros y directorios, el desarrollo de aplicaciones que acceden a bases de datos relacionales, el desarrollo de aplicaciones que hacen uso de bases de datos orientadas a objetos, el desarrollo de aplicaciones que acceden a bases de datos XML y el desarrollo de componentes de acceso a datos y su integración en aplicaciones.
