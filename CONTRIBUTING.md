# 🤝 Guía de Contribución - Comunidad DAM/DAW

¡Qué bueno que te hayas animado a colaborar! Este repositorio crece gracias a personas como tú. Para que todo funcione como la seda y el material sea útil para todos, sigue estas pequeñas pautas.

---

## 🚀 ¿Cómo puedo colaborar?

Puedes aportar de muchas formas:
* **Subiendo tus tareas y prácticas:** Para que otros vean diferentes soluciones.
* **Compartiendo apuntes o recursos:** En la carpeta `General` de cada asignatura.
* **Corrigiendo errores:** Si ves una errata en un ejercicio o un bug en un código.
* **Mejorando la documentación:** Añadiendo archivos `README.md` con enunciados.

---

## 📂 Estructura del Repositorio

Es vital mantener el orden para que nadie se pierda:

1.  **Asignaturas:** Busca la carpeta de la asignatura correspondiente (ej: `PROG`, `BD`, `DWES`).
2.  **Material General:** Si quieres subir recursos de estudio, usa la carpeta `General` dentro de la asignatura.
3.  **Tus Tareas:** Crea una carpeta con el formato `Año-Curso/TuNick` (ej: `2024-2025/tu-usuario`).
4.  **Enunciados:** Si subes una tarea, intenta incluir un pequeño `README.md` con el enunciado para que se entienda el contexto.

---

## 🛠️ Pasos para subir tu contenido

### Opción A: Desde tu PC (Git)
1. **Clona** el repositorio: `git clone https://github.com/FPDRioja/DAM_DAW.git`
2. **Crea una rama** para tus cambios: `git checkout -b tu-nombre/descripcion`
3. **Añade tus archivos** siguiendo la estructura mencionada.
4. **Haz commit** con un mensaje descriptivo: `git commit -m "Añadida tarea 3 de PROG - [TuNick]"`
5. **Sube los cambios** y abre un **Pull Request**.

### Opción B: Desde la web de GitHub
1. Haz un **Fork** de este repositorio.
2. Sube tus archivos o crea carpetas directamente en tu fork.
3. Haz el commit explicando los cambios.
4. Crea un **Pull Request** hacia el repositorio principal.
